# CSV File Management Agent — Core Instructions

## Project Overview
This agent automates a full CSV file management pipeline:
1. Generates a list of numbers (1–100)
2. Splits it into 10 sublists
3. Writes each sublist to individual CSV files
4. Organizes files into 10 folders via circular combination logic
5. Processes each CSV, computes squares, and updates `master.csv`
6. Recovers missing files by searching across folders

## Key Rules
- Always use `exist_ok=True` when creating folders to prevent crashes
- Always check if a file exists before reading, moving, or deleting it
- Use circular indexing (`% len`) when assigning files to folders
- Log every action taken (creation, copy, deletion, update)
- If a file is not found in the primary folder, search all other folders before failing
